# 📊 FoodHub Data Analysis – Business Insights 

## 📌 Overview

Exploratory data analysis project focused on identifying business insights from a food delivery dataset.

Analyze customer behavior and demand patterns to support business decisions.

## 🔍 Key Analysis

* Order trends
* Customer behavior
* Pricing insights
* Delivery performance

## 📊 Insights

* Identified key factors influencing order volume
* Highlighted opportunities for pricing optimization

## 🛠 Tools

Python, Pandas, Matplotlib, Seaborn
